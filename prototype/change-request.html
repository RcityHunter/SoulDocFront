## 认证方式
SoulDoc 支持以下认证方式：
- - JWT Token 认证
- - OAuth 2.0
+ - JWT Token 认证（主要方式）
+ - OAuth 2.0（第三方登录）
+ - OIDC（企业级身份认证）
+ - Rainbow-Auth 集成（内部系统）
+ ### OIDC 配置说明
+ OIDC 登录需要配置 `OIDC_ISSUER`, `OIDC_CLIENT_ID`, `OIDC_CLIENT_SECRET`...
