1. 产品定位

SoulDoc 是一套面向 AI 原生内容生产、知识组织、团队协作与多角色共创的文档/知识系统。以 Space / Doc / Version / Tag / File / Member 为核心对象模型，以 AI 协作、知识沉淀、搜索召回、权限隔离、多场景发布为关键能力。

💡 核心价值：SoulDoc 不是单纯"写文档"的工具，而是一个面向 AI 时代的"内容资产生产与管理平台"。

2. 核心功能模块

🗂️
空间系统
多独立知识空间，支持公开/私有模式
📄
文档系统
树状结构、Markdown编辑、版本管理
✨
AI 能力层
5层AI能力：写作→结构化→协作→检索→运营
🔀
变更请求
Git式内容协作流：草稿→审核→合并

3. AI 能力层级设计

L1辅助写作
标题生成、摘要生成、文本润色、多风格改写、段落扩写/压缩
L2知识结构化
自动生成大纲、识别章节层级、提取标签、生成FAQ、生成术语表
L3协作生产
多角色AI工作流：创意总监AI、编辑AI、审校AI、SEO AI、翻译AI
L4知识检索增强 (RAG)
语义问答、引用文档片段、推荐阅读、知识导航
L5自动化运营
自动更新SEO字段、监控内容缺失、发现陈旧文档、生成周报
