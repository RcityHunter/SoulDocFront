1. 产品定位

SoulDoc 是一套面向 AI 原生内容生产、知识组织、团队协作与多角色共创的文档/知识系统。以 Space / Doc / Version / Tag / File / Member 为核心对象模型，以 AI 协作、知识沉淀、搜索召回、权限隔离、多场景发布为关键能力。

SoulDoc 不是单纯"写文档"的工具，而是一个面向 AI 时代的"内容资产生产与管理平台"。

2. 核心功能模块

2.1 空间系统 (Space)

  • 创建多个独立知识空间
  • 支持公开空间与私有空间
  • 支持空间描述、Slug、品牌信息、主题配置
  • 多租户能力的承载单位

2.2 文档系统 (Document)

  • 文档树结构 / 章节层级
  • Markdown 编辑与实时预览
  • 封面、摘要、Slug、可见性设置
  • 关联标签、附件、评论
